    lane=0 就是任意读，
    lane=1 就是任意写。

    • config
    • params
    • residual_labels
    • input_encoder
    • output_encoder

    你会看到很多这种名字：
    • count_True_*
    • length_*
    • selector_width_*
    • sequence_map_*
      这说明它不是“黑盒神经网络”，而是把一组离散逻辑规则编译成 Transformer 了。

    常见做法是：
    • 找每个 sequence_map_* / map_* 在比较什么位置
    • 找 selector_width_* 在统计哪一组 token
    • 找 count_True_* 最终在数多少个条件成立
      一般会还原成“16 个格子之间的约束关系”。
  6. 把约束翻成方程/脚本去解。
    因为输入正好是 16 位、字符集只有 1-4，通常就是一个 4x4 格子题。
    你把模型里的比较关系整理成：
    • 哪些位置必须相等
    • 哪些位置必须不等

1.reservation

  1. load_dotenv()
    从当前目录的 .env 文件中读取键值对，写入进程环境变量（os.environ）。
  2. PROMPT = os.getenv("PROMPT", "bananananannaanan")
    • os.getenv("PROMPT")：从操作系统环境中读取名为 PROMPT 的变量。
    • 如果环境里没有这个变量，才会使用后面的默认值 "bananananannaanan"。
  3. response = client_socket.recv(...).decode().strip()
    收到你在 nc 里输入的一行内容，去掉回车换行，得到一个字符串 response。
  4. if response == PROMPT:
    比较输入的字符串和环境变量 **PROMPT** 当前的值是否完全相同（逐字符一致）。
  5. 要通过验证，必须发送「和服务器进程环境变量 PROMPT 的值完全一样的字符串」。

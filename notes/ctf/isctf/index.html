欧拉定理：

$ msg^{ϕ(N)}≡1\ mod\ N $

而 RSA 中：

$ ϕ(N)=(p−1)(q−1)=N−p−q+1 $

所以我们有：

$ msg^{N−p−q+1}≡1\ mod\ N $

两边同时乘以 msg_p_+q−1，得：

$ msg^{N+1}≡msg^{p+q}\ mod\ N→ ct2≡msg^{N+1}\ mod\ N $

$ ct1≡msg^{e}\ mod\ N $

计算一下发现：

$ gcd(e,N+1)=1 $

这样可以利用gcdext算出

$ ae+b(N+1)=1 $

有：

$ msg^1≡msg^{ea+b(N+1)}≡(msg^e)^a*(msg^{N+1})^b≡(ct1)^a*(ct2)^b\ \ mod\ N $
